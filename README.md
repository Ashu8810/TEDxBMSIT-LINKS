# TEDxBMSITM Cinematic Links

A premium, cinematic landing page for TEDxBMSITM, designed with a dark luxury stage aesthetic.

## 🚀 Features

- **Dark Luxury Aesthetic**: Pure black environment with deep red ambient glows.
- **Cinematic Lighting**: Volumetric spotlight system with three top spotlights.
- **Official Branding**: Integrated official TEDxBMSITM logo and institutional crest.
- **Interactive Socials**: Glassmorphism pill buttons for Instagram, X (Twitter), and LinkedIn.
- **Immersive Atmosphere**: Animated dust particles and bottom horizon floor reflections.
- **Performance**: Built with Next.js 16, Tailwind CSS, and Framer Motion.

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) / Custom SVGs

## 📦 Deployment

This project is optimized for deployment on **Vercel**.

1. Push your code to GitHub.
2. Import the repository into your Vercel Dashboard.
3. Vercel will automatically detect the Next.js framework and deploy.

## 💻 Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
